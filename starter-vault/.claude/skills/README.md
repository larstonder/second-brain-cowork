# starter-vault/.claude/skills/

The skills bundled with this starter pack. **In Claude Cowork they are not picked up automatically from this folder** (that is a Claude Code behavior). They are pre-zipped under `skills-to-upload/` in the repo root, and you upload them once via **Customize ▸ Skills ▸ + ▸ upload a skill (ZIP)**, then toggle them on.

| Skill | Used for |
|-------|----------|
| `obsidian-vault` | Core vault conventions (frontmatter, linking, folder layout). Upload this one at minimum. |
| `youtube-to-note` | Challenge 1: turn a YouTube video into a Learning note. Works by pasting the transcript, no terminal or downloads. |
| `skill-creator` | Challenge 2: scaffolding your own skill. Optional. |
| `brainstorming` | Challenge 2: thinking through what to build. Optional. Sourced from [obra/superpowers](https://github.com/obra/superpowers). |

This folder is kept as the source of truth for the skills (and so they still work if anyone opens the vault in Claude Code, which does auto-discover `.claude/skills/`). For Cowork, use the ZIPs.
